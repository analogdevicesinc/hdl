Help & Support
--------------

* The `carriers (abbrevations can be found here) <http://wiki.analog.com/resources/fpga/docs/hdl#list_of_carriers_supported>`_ are commonly available FPGA evaluation boards.
* The `HDL user guide <http://wiki.analog.com/resources/fpga/docs/hdl>`_ contains all the documentation, build instructions and register map tables.
* The following quick links allows you to browse the github repository for a list of current `branches <http://github.com/analogdevicesinc/hdl/branches/active>`_, :git-hdl:`library components <library>`, and :git-hdl:`projects <projects>`.
* Questions? We can help with `FPGA questions <https://ez.analog.com/community/fpga>`_, `Linux driver questions <https://ez.analog.com/community/linux-device-drivers/linux-software-drivers>`_, `No-OS Drivers questions <https://ez.analog.com/community/linux-device-drivers/microcontroller-no-os-drivers>`_.
