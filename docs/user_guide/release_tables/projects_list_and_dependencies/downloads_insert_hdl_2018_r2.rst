* The following quick links allows you to browse the github repository for a list of current `branches <http://github.com/analogdevicesinc/hdl/branches/active>`_, :git-hdl:`library components <master:library>`, and :git-hdl:`projects <master:projects>`.
* Questions? We can help with `FPGA questions <https://ez.analog.com/community/fpga>`_, `Linux driver questions <https://ez.analog.com/community/linux-device-drivers/linux-software-drivers>`_, `No-OS Drivers questions <https://ez.analog.com/community/linux-device-drivers/microcontroller-no-os-drivers>`_.
