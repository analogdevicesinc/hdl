# PULSAR-ADC HDL Project

## Building the project

:warning: Make sure that you set up your required ADC resolution in [../common/pulsar_adc_bd.tcl](../common/pulsar_adc_bd.tcl)

```
cd projects/pulsar_adc/coraz7s
make
```
